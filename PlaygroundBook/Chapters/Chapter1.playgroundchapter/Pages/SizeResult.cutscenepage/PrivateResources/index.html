Larger Size = Slower


In physics, mass = volume * density. Increasing the size of a marble means there is more mass to move when rolling. This lowers the speed. It also increases the sustained speed. Sustained speed doesn't affect our marble run as much as acceleration.

We can simulate volume in SpriteKit using the initializer of SKPhysicsBody:

SKPhysicsBody(circleOfRadius: CGFloat)

Mass and Density can also be configured on an existing SKPhysicsBody:

SKPhysicsBody.mass = CGFloat
SKPhysicsBody.density = CGFloat
